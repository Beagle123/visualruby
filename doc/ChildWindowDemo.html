#buttonOpenModal__clicked(*argv) ⇒ Object

This modal window has its “modal” property set to true in glade. I pass a reference to “self” in the show method, setting the parent to this window, making it always on top. Its a good idea to have modal windows always on top because the parent is frozen, so the next action must be in the modal window.
